# Personal Expense Tracker

A full-stack expense tracking app built with Spring Boot and React. Users can create, edit, delete, filter, sort, and summarize expense.

## Teck Stack

- Backend: Spring Boot
- frontend: React + Vite
- API: REST
- Styling: CSS
- Database: 

## Features

- Add, edit, and delete expenses
- Filter expenses by newest or oldest date
- View total spending summary
- View category totals
- Form validation and user feedback
- Loading, error, and empty states
- Configurable frontend API URL with Vite environment variables

## Environment Variables

Copy `.env.example` to `.env` if you need to configure the backend URL:

```env
VITE_API_URL=http://localhost:8080/api/expenses
```
