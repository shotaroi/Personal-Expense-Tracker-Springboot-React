## Environment Variables

Copy `.env.example` to `.env` if you need to configure the backend URL:

```env
VITE_API_URL=http://localhost:8080/api/expenses
```

## Run Locally

Install dependencies:

```bash
npm install
```

Start the Vite dev server:

```bash
npm run dev
```

## Useful Scripts

Run lint checks:

```bash
npm run lint
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
